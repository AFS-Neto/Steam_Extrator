# Steam Extractor

## Overview
Steam Extractor is a tool designed to extract and manage Steam game data and information.

## Installation
1. Clone or download the repository
2. Install required dependencies
3. Run the program from your terminal

## Usage

### Configuration
Set up your Steam directory path and API credentials in the config file before running the tool.

### Game Data Extraction
Extracts metadata, achievements, and game information from your Steam library.

### Export Options
Generates reports and exports data in multiple formats (JSON, CSV, XML, Sqlite).

### TLS Trusted Layer
Run the els_db_trusted.py file to creat and upsert values from raw data

### Filtering
Filter games by genre, playtime, achievement status, or custom criteria.

## Troubleshooting
- Ensure Steam is properly installed
- Verify API access permissions
- Check log files for detailed error messages
